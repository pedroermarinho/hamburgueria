# hamburgueria

> Teste para vaga de Desenvolvedor de Software Junior 

` Esse projeto teve ênfase no backend `

### Dependência 
- Java 11
- PostgreSQL
- Node >=14
- Vuejs ~= 2
- Yarn

### Instalação das dependência do projeto 

/serve
``` bash 
./mvnw spring-boot:install
```
/web 
``` bash 
yarn install
```

### Rodando o projeto 

/serve 
``` bash 
./mvnw spring-boot:run
```
/web 
``` bash
yarn serve --port 8081
```
